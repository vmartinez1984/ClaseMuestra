XAML

eXtensible Application Markup Languaje, basado en XML. Es el lenguaje que nos permite construir la vista en la aplicación, páginas, controles, contenedores.

Cuando creas tu proyecto proyecto mediante VS, crea dos archivos un .xaml el cual tiene etiquetas tipo html o como dice la definición xml y otro archivo del mismo nombre .xaml.cs donde se hace la programación en C#. Como se puede apreciar en la siguiente imágen:

Para este primer acercamiento, revisaremos el MainPage.xml

Ejercicio de muestra
Realizar un formulario simple, en la vista tendra una etiqueta, una caja de texto y un botón. El botón en su evento clicked, tomara el contenido de la caja de texto y lo mostrara en un modal. Este formulario recogera el nombre.
Ejercicio de clase 01, Inicio de sesión v1
Realice un formulario de incio de sesión, realice una pequeña validación para que se muetre bienvenido o que las credenciales no son validas.
Ejercicio de clase 02, Captura de datos de cliente
Realice un formulario de captura de datos, el formulario debera recibir:
Campo Tipo Obligatorio Min Max Ejemplo
Nombre Letras Si 2 30 María
Apellidos Letras Si 2 50 Hernández Hernández
Telefono Solo números Si 2 50 5556581111
Correo Correo Si 10 50 maria.hdz@gmail.com
Contraseña Letras, números y especiales Si 6 12 Mary123
Rfc Letras y números Si 13 13 HEHM9005056TA
Realice validaciones simples, solo numeros, que no este vacio el campo, segun considere.
